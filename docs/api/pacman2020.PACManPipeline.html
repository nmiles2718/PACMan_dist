class pacman2020.PACManPipeline(cycle=None, model_name='')¶

Bases: utils.tokenizer.PACManTokenizer

Parameters

  • cycle (int) – The HST Cycle number to analyze

  • model_name (str) – The name of one of the trained models in the ~/PACman_dist/models directory.
