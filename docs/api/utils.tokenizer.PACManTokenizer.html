PACManTokenizer¶

class utils.tokenizer.PACManTokenizer¶

Bases: object

This class contains the text pre-processing functionality

The tokenizer object leverages the powerful NLP model, spaCy, to perform the lemmatization and tokenization of each proposal’s text.
