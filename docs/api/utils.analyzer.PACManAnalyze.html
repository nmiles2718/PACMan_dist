compute_accuracy_measurements(df=None, normalize=False)¶

Compute the classification accuracies

Compute the standard accuracy and a custom accuracy. The standard accuracy is the number of proposals where the correct category is the most probable. The custom accuracy is the number of proposals where the correct category is in the top two most probable classes.
