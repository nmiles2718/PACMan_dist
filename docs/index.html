pacman2020¶

This package is designed to handle the classification of text data scraped from HST proposals. It contains all of the functionality required to build an end-to-end machine learning pipeline for the classification of HST Proposals into one of the HST science categories. In short, we provide tools for

  1. proposal scraping (HSTProposalScraper),

  2. text preprocessing (PACManTokenizer) using spaCy,

  3. training and testing on hand classified proposals (PACManTrain),

  4. classification of unclassified proposals (PACManPipeline).
